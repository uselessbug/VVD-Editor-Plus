Help

What is VVD

VVD is a file under the same path as .ddb and .ddi files, using the same name as .ddb and .ddi files, storing the basic information about the voicebank.

What can we do by editing VVD

Editing VOICENAME in VVD can change the name of a voicebank in V2/3/4 editor. (V5 editor reads the name string in registry and won't be affected.)

How to use the install script

This script can install a voicebank into V4

  1. Fill in the CompID and the name of a voicebank, download the.vvd file and the script.
  2. Move the .ddb, .ddi, and .vvd file under a subdir whose name is same as the CompID of this voicebank. (E.g. Luotianyi_CHN_Meng: D:\Program Files (x86)\VoiceDB\LuoTianyiV4_CHN\BK8H76TAEHXWSKDB) Confirm .ddb, .ddi and .vvd files share the same name and under the same dir.
  3. Move install.bat into the same dir as above and Run as Administrator.
  4. Now the voicebank should have been successfully installed. Similarly, you can uninstall it using uninstall.bat.

About CompID

Since the install script is for V4, you need to get a CompID for the voicebank which matches its language. V4 voicebanks can directly use the origin CompID. V5 voicebanks, however, need to get a corresponding V4 CompID. Here are some V4 CHN CompID and V4 JPN CompID, click to pick it randomly.
